# Projeto da disciplina Bootcamp 1

# Portfólio Profissional — Paulo Marcos Neves de Alcântara

Este projeto consiste em um portfólio pessoal desenvolvido com foco em apresentar minha trajetória acadêmica, habilidades técnicas, cursos, projetos e objetivos profissionais na área de tecnologia.

O site foi construído utilizando HTML e CSS puro, com um design moderno, responsivo e intuitivo, buscando oferecer uma navegação simples e agradável em diferentes dispositivos.

---

# Objetivo do projeto

O principal objetivo deste projeto é criar uma presença profissional online, reunindo informações importantes sobre minha formação, conhecimentos e interesses na área de Ciência da Computação.

Além disso, o projeto também tem como finalidade:

* Praticar desenvolvimento web front-end;
* Aplicar conceitos de estruturação de páginas com HTML;
* Trabalhar estilização e responsividade utilizando CSS;
* Desenvolver uma interface visual moderna;
* Organizar informações profissionais de forma clara e objetiva.

---

# Tecnologias utilizadas

## HTML5

O HTML foi utilizado para estruturar toda a página do portfólio, organizando seções como:

* Página inicial;
* Sobre mim;
* Habilidades;
* Projetos;
* Cursos;
* Contato.

Também foi responsável pela inserção de links, textos, imagens e organização geral do conteúdo.

---

## CSS3

O CSS foi utilizado para toda a parte visual do projeto, incluindo:

* Layout responsivo;
* Cores e identidade visual;
* Animações e efeitos de hover;
* Organização dos elementos;
* Estilização dos cards;
* Responsividade para dispositivos móveis.

O objetivo foi criar uma interface moderna e agradável visualmente.
